7 คีย์? ไม่เป็นไร ขอบคุณ 

นี่คือโปรแกรมเล่น BMS ที่อ้างอิงจาก Pygame ซึ่งจะ **จัดเรียงใหม่แบบบังคับ** ทุกชาร์ต BMS ให้เป็น **4 คีย์ (DFJK)** เพื่อการเล่น

## ภาพรวมโปรเจกต์

Only4BMS ถูกสร้างขึ้นสำหรับผู้ที่รักใน 4 คีย์ ที่ต้องเผชิญกับชาร์ต BMS ที่ซับซ้อน เช่น 7 คีย์ หรือ 14 คีย์ แล้วคิดว่า "ฉันจะเล่นทั้งหมดนี้ได้อย่างไร?"

โปรเจกต์นี้ใช้ Pygame เพื่อแยกวิเคราะห์ไฟล์ BMS และเตรียมสภาพแวดล้อมที่คุณสามารถเพลิดเพลินกับชาร์ตทุกรูปแบบ **ที่ถูกจับคู่ใหม่เป็น 4 เลน** พร้อมกับเสียงคีย์ซาวด์

## คุณสมบัติหลัก

**การจับคู่ 4 คีย์แบบบังคับ**: กำหนดชาร์ต 5 คีย์, 7 คีย์, 10 คีย์ และ 14 คีย์ ไปยัง 4 เลน (D, F, J, K) โดยอัตโนมัติโดยใช้อัลกอริทึมทางคณิตศาสตร์

**เครื่องมือตรวจสอบความหนาแน่น**: ตรวจสอบและแสดงภาพความหนาแน่นของโน้ตเมื่อชาร์ต 7 คีย์ ถูกรวมเข้าเป็น 4 คีย์

## 🛠️ เทคโนโลยีที่ใช้

**ภาษา**: Python 3.x

**ไลบรารี**: Pygame (เสียงและการแสดงผล)

**รูปแบบ**: Be-Music Script (.bms, .bme, .bml)

## วิธีการเล่น (การเพิ่มเพลงและการรัน)

1. **รับตัวเกม**: ดาวน์โหลด `Only4BMS.exe` ที่สร้างไว้ล่วงหน้า หรือสร้างด้วยตนเอง
2. **เปิดเกม**: เพียงแค่รัน `Only4BMS.exe` จากโฟลเดอร์ใดก็ได้ 
3. **โฟลเดอร์ `bms`**: เมื่อเปิดเกม เกมจะสร้างโฟลเดอร์ `bms` โดยอัตโนมัติในไดเรกทอรีเดียวกับไฟล์รัน หากไม่พบเพลง เกมจะสร้างเพลงเดโมพื้นฐานขึ้นมา
4. **การเพิ่มเพลงของคุณเอง**: 
   - ดาวน์โหลดไฟล์ BMS/BME/BML และสื่อที่เกี่ยวข้อง (เสียง/วิดีโอ BGA)
   - แตกไฟล์ลงในโฟลเดอร์ย่อยของตนเองภายในไดเรกทอรี `bms/`
   - ตัวอย่างโครงสร้าง:
     ```text
     [ไดเรกทอรีที่มี Only4BMS.exe]
     ├── Only4BMS.exe
     └── bms/
         ├── Awesome Track/
         │   ├── song.bms
         │   ├── audio.wav
         │   └── video.mp4
         └── Another Track/
             └── ...
     ```

## 🎵 โหมดคอร์ส (Roguelike การฝึกซ้อม)

เบื่อเพลงเดิมๆ หรือเปล่า? เข้าสู่ **โหมดคอร์ส** โหมดการฝึกซ้อมที่สร้างขึ้นใหม่ทุกครั้ง!
- สร้างแผนภูมิแบบสุ่มที่แตกต่างกันทุกครั้งที่เล่น
- ระดับความยาก: ระดับเริ่มต้น (BPM 80~110), ระดับกลาง (BPM 120~160), ระดับสูง (BPM 160~200)
- พบกับลองโน้ตและการเปลี่ยนความเร็ว (Gimmicks) เมื่อระดับความยากเพิ่มขึ้น
- แต่ละด่านใช้เวลาประมาณ 30 วินาที เอาชีวิตรอดเพื่อดูว่าจะเกิดอะไรขึ้นต่อไป!

## 🤖 การฝึก AI และโหมดผู้เล่นหลายคน

Only4BMS มีโหมด AI Multiplayer ที่ขับเคลื่อนโดย **การเรียนรู้แบบเสริมกำลัง (Reinforcement Learning - PPO)**

### วิธีการทำงาน
- AI ถูกฝึกโดยใช้ `stable-baselines3` บนแทร็กจังหวะที่สร้างขึ้นตามขั้นตอนและเพลงเดโมที่ไม่มีลิขสิทธิ์
- **การปฏิบัติตามกฎหมาย**: เพื่อให้แน่ใจว่าได้มาตรฐานทางจริยธรรม รุ่นที่เผยแพร่อย่างเป็นทางการจะมาพร้อมกับโมเดลที่ฝึก *โดยเฉพาะ* จากข้อมูลที่ไม่ใช่เชิงพาณิชย์ในระหว่างกระบวนการ CI/CD
- **ระดับความยาก**:
   - **NORMAL**: ฝึกมา 25,000 ขั้นตอน มีความแม่นยำสูงแต่มีความผิดพลาดเป็นครั้งคราวเหมือนมนุษย์
   - **HARD**: ฝึกมา 40,000 ขั้นตอน จังหวะและการรักษาคอมโบเกือบจะสมบูรณ์แบบ

### การฝึกในเครื่อง (Local Training)
หากคุณต้องการฝึกโมเดลของคุณเองในเครื่อง:
1. ติดตั้งสิ่งที่จำเป็น: `pip install stable-baselines3 shimmy gymnasium torch`
2. รันสคริปต์ฝึก: `python -m only4bms.ai.train`
3. ไฟล์ที่ได้คือ `model_normal.zip` และ `model_hard.zip` จะถูกบันทึกใน `src/only4bms/ai/`

### ระบบ CI/CD อัตโนมัติ
เวิร์กโฟลว์ GitHub Actions ของเราจะฝึกโมเดล AI ใหม่โดยอัตโนมัติจากจุดเริ่มต้นโดยใช้ `Mock Song Demo` สำหรับทุกการเปิดตัว เพื่อให้แน่ใจว่าตัวเกมที่แจกจ่ายให้กับผู้ใช้นั้น "สะอาด" และได้รับการปรับปรุงอยู่เสมอ
